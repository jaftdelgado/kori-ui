## API Reference

The `Switch` component uses Svelte 5 runes for reactivity and state binding.

| Prop          | Type                         | Default     | Description                                     |
| :------------ | :--------------------------- | :---------- | :---------------------------------------------- |
| `checked`     | `boolean`                    | `false`     | **$bindable**. The current state of the switch. |
| `size`        | `'sm' \| 'md' \| 'lg'`       | `'md'`      | Sets the scale of the track and thumb.          |
| `disabled`    | `boolean`                    | `false`     | Disables interaction and dims the opacity.      |
| `label`       | `string`                     | `undefined` | Main text displayed next to the switch.         |
| `description` | `string`                     | `undefined` | Supplementary text displayed under the label.   |
| `onchange`    | `(checked: boolean) => void` | `undefined` | Callback fired when the toggle state changes.   |
