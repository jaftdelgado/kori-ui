## API Reference

The `ScrollArea` component uses Svelte 5 runes and ResizeObserver for dynamic scroll state detection.

| Prop            | Type                         | Default      | Description                                             |
| :-------------- | :--------------------------- | :----------- | :------------------------------------------------------ |
| `orientation`   | `'vertical' \| 'horizontal'` | `'vertical'` | The scroll direction of the container.                  |
| `hideScrollBar` | `boolean`                    | `false`      | Whether to hide the native scrollbar.                   |
| `size`          | `number`                     | `40`         | The size (in pixels) of the scroll shadow indicator.    |
| `offset`        | `number`                     | `0`          | Offset threshold for detecting scroll start/end states. |
| `isEnabled`     | `boolean`                    | `true`       | Whether scroll state detection is active.               |
| `class`         | `string`                     | `undefined`  | Additional CSS classes to apply to the container.       |
