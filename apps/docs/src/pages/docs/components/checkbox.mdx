## API Reference

The `Checkbox` component uses Svelte 5 runes for reactivity and state binding.

| Prop          | Type                         | Default     | Description                                       |
| :------------ | :--------------------------- | :---------- | :------------------------------------------------ |
| `checked`     | `boolean`                    | `false`     | **$bindable**. The current state of the checkbox. |
| `label`       | `string`                     | `undefined` | Main text displayed next to the checkbox.         |
| `description` | `string`                     | `undefined` | Supplementary text displayed under the label.     |
| `disabled`    | `boolean`                    | `false`     | Disables interaction and dims the opacity.        |
| `name`        | `string`                     | `undefined` | The name attribute for form submission.           |
| `onchange`    | `(checked: boolean) => void` | `undefined` | Callback fired when the checked state changes.    |
