# Radio Group

A set of checkable buttons—known as radio buttons—where no more than one button can be checked at a time.

## Examples

### Sizes

Radio buttons support `sm`, `md`, and `lg` sizes which are inherited from the `RadioGroup`.

<ComponentExample code={RadioSizesCode}>
  <RadioSizes client:load />
</ComponentExample>

### Orientation

By default, the orientation is `vertical`, but it can be set to `horizontal`.

<ComponentExample code={RadioOrientationCode}>
  <RadioOrientation client:load />
</ComponentExample>

### Disabled

Radios can be disabled individually or by disabling the entire group.

## API Reference

### RadioGroup

| Prop          | Type                         | Default      |
| :------------ | :--------------------------- | :----------- |
| `value`       | `any`                        | `undefined`  |
| `name`        | `string`                     | `UUID`       |
| `size`        | `"sm" \| "md" \| "lg"`       | `"md"`       |
| `orientation` | `"horizontal" \| "vertical"` | `"vertical"` |
| `disabled`    | `boolean`                    | `false`      |
| `onchange`    | `(value: any) => void`       | `undefined`  |

### Radio

| Prop          | Type      | Default      |
| :------------ | :-------- | :----------- |
| `value`       | `any`     | **Required** |
| `label`       | `string`  | `undefined`  |
| `description` | `string`  | `undefined`  |
| `disabled`    | `boolean` | `false`      |
