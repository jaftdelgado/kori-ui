## API Reference

The `Accordion` component uses a data-driven approach with `items` and Svelte 5 snippets for custom rendering.

| Prop                  | Type                     | Default     |
| :-------------------- | :----------------------- | :---------- |
| `items`               | `AccordionItemProps[]`   | `[]`        |
| `selectionMode`       | `"single" \| "multiple"` | `"single"`  |
| `defaultExpandedKeys` | `string[]`               | `[]`        |
| `children`            | `Snippet<[item]>`        | `undefined` |
| `titleContent`        | `Snippet<[item]>`        | `undefined` |
| `itemClasses`         | `object`                 | `{}`        |

### AccordionItemProps

| Property       | Type      | Description                                     |
| :------------- | :-------- | :---------------------------------------------- |
| `key`          | `string`  | Unique identifier for the item.                 |
| `title`        | `string`  | Main text shown in the trigger.                 |
| `subtitle`     | `string`  | Secondary text shown below the title.           |
| `disabled`     | `boolean` | Whether the item is interactive.                |
| `startContent` | `Snippet` | Content shown before the title (e.g., an icon). |
| `indicator`    | `Snippet` | Custom icon to replace the default chevron.     |
