## API Reference

### Avatar

| Prop       | Type                   | Default    | Description                                                     |
| :--------- | :--------------------- | :--------- | :-------------------------------------------------------------- |
| `src`      | `string`               | `undefined` | The source URL of the image to display.                        |
| `alt`      | `string`               | `"Avatar"` | Accessible text for the image.                                  |
| `size`     | `"sm" \| "md" \| "lg"` | `"md"`     | The scale of the avatar.                                        |
| `fallback` | `string`               | `"?"`      | Text to display if the image fails to load.                     |
| `grouped`  | `boolean`              | `false`    | Adds a ring around the avatar, useful when used in an `AvatarGroup`. |
| `class`    | `string`               | `""`       | Additional CSS classes for the container.                       |

### AvatarGroup

| Prop       | Type                   | Default    | Description                               |
| :--------- | :--------------------- | :--------- | :---------------------------------------- |
| `children` | `Snippet`              | `undefined` | The avatars to display in the group.      |
| `class`    | `string`               | `""`       | Additional CSS classes for the container. |
