## API Reference

| Prop       | Type                   | Default    | Description                                 |
| :--------- | :--------------------- | :--------- | :------------------------------------------ |
| `src`      | `string`               | `""`       | The source URL of the image to display.     |
| `alt`      | `string`               | `"Avatar"` | Accessible text for the image.              |
| `size`     | `"sm" \| "md" \| "lg"` | `"md"`     | The scale of the avatar.                    |
| `fallback` | `string`               | `"?"`      | Text to display if the image fails to load. |
| `class`    | `string`               | `""`       | Additional CSS classes for the container.   |
