## API Reference

| Prop          | Type                         | Default        | Description                                    |
| :------------ | :--------------------------- | :------------- | :--------------------------------------------- |
| `options`     | `SegmentedOption[]`          | `[]`           | Array of options to render.                    |
| `value`       | `string`                     | `undefined`    | **$bindable**. Value of the selected segment.  |
| `size`        | `"md" \| "lg"`               | `"md"`         | The vertical scale of the component.           |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | The layout direction of the segments.          |
| `disabled`    | `boolean`                    | `false`        | Disables all interactions with the component.  |
| `onchange`    | `(value: string) => void`    | `undefined`    | Callback fired when a new segment is selected. |
