## API Reference

| Prop          | Type                   | Default     | Description                                  |
| :------------ | :--------------------- | :---------- | :------------------------------------------- |
| `value`       | `string`               | `""`        | **$bindable**. The numeric string value.     |
| `length`      | `number`               | `6`         | Total number of input slots.                 |
| `separators`  | `number[]`             | `[]`        | Indices after which to render a separator.   |
| `size`        | `"sm" \| "md" \| "lg"` | `"md"`      | The scale of the input slots.                |
| `disabled`    | `boolean`              | `false`     | Disables all input slots.                    |
| `placeholder` | `string`               | `undefined` | Optional placeholder string (per character). |
| `label`       | `string`               | `undefined` | Label text displayed above the inputs.       |
| `description` | `string`               | `undefined` | Helper text displayed below the inputs.      |
| `isRequired`  | `boolean`              | `false`     | Adds a visual indicator to the label.        |
