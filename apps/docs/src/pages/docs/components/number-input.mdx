## API Reference

The `NumberInput` component uses Svelte 5 runes for reactivity and state binding.

| Prop          | Type                   | Default          |
| :------------ | :--------------------- | :--------------- |
| `value`       | `number \| null`       | `null`           |
| `size`        | `'sm' \| 'md' \| 'lg'` | `'md'`           |
| `label`       | `string`               | `undefined`      |
| `description` | `string`               | `undefined`      |
| `isRequired`  | `boolean`              | `false`          |
| `min`         | `number`               | `-Infinity`      |
| `max`         | `number`               | `Infinity`       |
| `step`        | `number`               | `1`              |
| `disabled`    | `boolean`              | `false`          |
| `placeholder` | `string`               | `undefined`      |
| `id`          | `string`               | `auto-generated` |
