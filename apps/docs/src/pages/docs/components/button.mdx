## API Reference

The `Button` component accepts the following properties through Svelte 5 runes.

| Prop           | Type                                  | Default     |
| :------------- | :------------------------------------ | :---------- |
| `variant`      | `'primary' \| 'secondary' \| 'ghost'` | `'primary'` |
| `size`         | `'sm' \| 'md' \| 'lg'`                | `'md'`      |
| `disabled`     | `boolean`                             | `false`     |
| `startContent` | `Snippet \| Component \| string`      | `undefined` |
| `endContent`   | `Snippet \| Component \| string`      | `undefined` |
