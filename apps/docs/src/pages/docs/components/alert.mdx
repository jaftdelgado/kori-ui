## API Reference

The `Alert` component accepts the following properties through Svelte 5 runes.

| Prop       | Type                                                             | Default     | Description                               |
| :--------- | :--------------------------------------------------------------- | :---------- | :---------------------------------------- |
| `variant`  | `'default' \| 'destructive' \| 'success' \| 'warning' \| 'info'` | `'default'` | The visual style of the alert.            |
| `title`    | `string \| Snippet`                                              | `undefined` | The title of the alert.                   |
| `icon`     | `Snippet \| Component \| string`                                 | `undefined` | Icon displayed at the start of the alert. |
| `children` | `Snippet`                                                        | `undefined` | The content description of the alert.     |
