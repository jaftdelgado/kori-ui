## API Reference

The `Separator` component uses Svelte 5 runes for reactive properties.

| Prop          | Type                         | Default        |
| :------------ | :--------------------------- | :------------- |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` |
| `decorative`  | `boolean`                    | `true`         |
| `label`       | `string`                     | `undefined`    |
| `class`       | `string`                     | `undefined`    |
