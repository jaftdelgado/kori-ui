## API Reference

The `Chip` component supports both a `label` prop and a `children` snippet for more complex content.

| Prop       | Type                              | Default     | Description                                       |
| :--------- | :-------------------------------- | :---------- | :------------------------------------------------ |
| `label`    | `string`                          | `undefined` | The text to display inside the chip.              |
| `variant`  | `"solid" \| "outline" \| "ghost"` | `"solid"`   | The visual style of the chip.                     |
| `size`     | `"sm" \| "md" \| "lg"`            | `"md"`      | The vertical scale (matching Switch heights).     |
| `disabled` | `boolean`                         | `false`     | Reduces opacity and prevents interaction.         |
| `onremove` | `() => void`                      | `undefined` | Displays a close icon and triggers this callback. |
