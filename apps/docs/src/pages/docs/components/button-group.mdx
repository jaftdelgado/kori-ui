## API Reference

The `ButtonGroup` component uses Svelte 5 runes and context to share properties with child buttons.

| Prop       | Type                                                                      | Default     | 
| :--------- | :------------------------------------------------------------------------ | :---------- | 
| `variant`  | `'primary' \| 'secondary' \| 'ghost' \| 'outline' \| 'danger' \| 'soft-danger'` | `'primary'` | 
| `size`     | `'sm' \| 'md' \| 'lg'`                                                    | `'md'`      | 
| `attached` | `boolean`                                                                 | `true`      | 
| `class`    | `string`                                                                  | `undefined` |
