## API Reference

The `Slider` component uses Svelte 5 runes and pointer events for a smooth experience.

| Prop          | Type                      | Default     | Description                                     |
| :------------ | :------------------------ | :---------- | :---------------------------------------------- |
| `value`       | `number`                  | `0`         | **$bindable**. The current value of the slider. |
| `min`         | `number`                  | `0`         | The minimum allowed value.                      |
| `max`         | `number`                  | `100`       | The maximum allowed value.                      |
| `step`        | `number`                  | `1`         | The increment between selectable values.        |
| `size`        | `"sm" \| "md" \| "lg"`    | `"md"`      | The visual scale of the component.              |
| `label`       | `string`                  | `undefined` | Label text displayed above the slider.          |
| `description` | `string`                  | `undefined` | Secondary text displayed under the label.       |
| `disabled`    | `boolean`                 | `false`     | Whether the slider is interactive.              |
| `onchange`    | `(value: number) => void` | `undefined` | Callback fired when the value changes.          |
