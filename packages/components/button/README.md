# @kori-ui/button

A versatile button component with multiple variants, sizes, and support for icons and content slots. Features ripple effects and smooth transitions for an enhanced user experience.

## Installation

```bash
npm install @kori-ui/button
# or
pnpm add @kori-ui/button
# or
bun add @kori-ui/button
```

## Usage

```svelte
<script>
  import { Button } from "@kori-ui/button";
</script>

<Button variant="primary" size="md">
  Click me
</Button>

<Button variant="secondary" size="lg" disabled>
  Disabled Button
</Button>

<Button variant="ghost" size="sm">
  Ghost Button
</Button>
```
